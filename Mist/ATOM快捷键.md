# ATOM快捷键

| Key          | Command          |
| ------------ | ---------------- |
| alt-1-9      | tab切换            |
| alt-\        | editor与project切换 |
| ctrl-shift-c | 拷贝当前文件路径         |
| ctrl-d       | 选择当前单词           |
| ctrl-j       | 链接行              |
| ctrl-shift-d | 复制该行             |
| ctrl-shift-k | 删除该行             |
| ctrl-up/down | 行上下移动            |
| ctrl-\       | slidebar的隐藏      |
| ctrl-shift-m | md视图             |
| ctrl-g       | 去行               |
